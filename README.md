# SnapStack.art

### A Little Startup

SnapStack.art, kullanıcıların kendi dijital sanat eserlerini paylaşabileceği ve keşfedebileceği bir platformdur. Hem yaratıcıları destekler hem de kullanıcıların eserleri kolayca keşfetmesini sağlar.

---

> Suppor by Cordision

## 🔹 Frontend

```json
{
    "Framework": "React Vite",
    "Langs": "Typescript, Tailwindcss, css",
    "Libs": "React Icons, MUI",
    "Modules": "Axios, ReactRouterDom",
    "Host": "Vercel.app"
}
````

### Özellikler

* Hızlı ve modern UI
* Responsive tasarım
* Dinamik sayfa yönlendirmeleri
* REST API ile veri yönetimi

## 🚀 Kurulum

### Frontend

```bash
git clone https://github.com/TPashaxrd/snapstack.art.git
cd frontend
npm install
npm run dev
```

### Backend

```bash
git clone https://github.com/TPashaxrd/snapstack.art.git
cd cd backend
npm install
npm run start
```

---

## 🛠️ Kullanılan Teknolojiler

* **Frontend:** React, Vite, TypeScript, TailwindCSS, MUI, Axios, React Router DOM
* **Backend:** Node.js, Express.js, MongoDB
* **Security:** express-rate-limit, cors, express-session, Cloudflare
* **Hosting:** Vercel (frontend), Cloudflare + VDS (backend)
